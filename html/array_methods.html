Methods that mutate the original array:
    push() - Adds elements to the end.
    pop() - Removes the last element.
    shift() - Removes the first element.
    unshift() - Adds elements to the beginning.
    splice() - Adds/removes elements at a specific index.
    sort() - Sorts the array in place.
    reverse() - Reverses the array in place.
    Methods that return a new array:
    map() - Creates a new array by applying a function to each element.
    filter() - Creates a new array with elements that pass a condition.
    concat() - Merges arrays and returns a new one.
    slice() - Extracts a portion and returns a new array.
    flat() - Flattens nested arrays.
    reduce() - Accumulates values, but doesn’t mutate the original array.
JavaScript Array Methods - Hinglish Mein 😎
   
        🔄 Mutating Methods (Jo Original Array Ko Change Kar Dete Hai)
    push() - Last mein element add karta hai.
    pop() - Last wala element nikal deta hai.
    shift() - First wala element nikal deta hai.
    unshift() - Starting mein element add karta hai.
    splice(start, deleteCount, ...items) - Beech mein se elements nikalta ya add karta hai.
    sort() - Array ko sort kar deta hai (ascending/descending).
    reverse() - Array ko ulta kar deta hai.
    fill(value, start, end) - Array ko ek hi value se bhar deta hai.
    copyWithin(target, start, end) - Ek part ko dusre jagah copy kar deta hai.
    
    🆕 Non-Mutating Methods (Original Array Ko Change Nahi Karte)
    concat() - Do ya zyada arrays ko jod kar nayi array banata hai.
    slice(start, end) - Array ka ek portion copy karta hai aur nayi array deta hai.
    map(callback) - Har element par function laga kar nayi array banata hai.
    filter(callback) - Condition pass karne wale elements ki nayi array banata hai.
    flat(depth) - Nested arrays ko ek level tak flatten kar deta hai.
    flatMap(callback) - Pehle map karta hai, fir flatten karta hai.
    
    🔍 Searching and Indexing Methods
    indexOf(value) - Pehli baar kaha mila, uska index deta hai.
    lastIndexOf(value) - Aakhri baar kaha mila, uska index deta hai.
    includes(value) - Check karta hai ki element exist karta hai ya nahi.
    find(callback) - Pehla element jo condition match kare, usko deta hai.
    findIndex(callback) - Pehle matching element ka index deta hai.
   
    🔢 Iteration Methods (Loop Wale)
    forEach(callback) - Har element par kuch operation karta hai.
    some(callback) - Ek bhi element match kare toh true deta hai.
    every(callback) - Har element match kare toh true deta hai.
    reduce(callback, initialValue) - Left se right tak reduce karke ek single value banata hai.
    reduceRight(callback, initialValue) - Right se left tak reduce karta hai.

    📝 String Conversion and Joining
    join(separator) - Sab elements ko jod kar ek string banata hai.
    toString() - Array ko string mein convert karta hai.
    toLocaleString() - Local format mein string banata hai.

    🗺️ Special Methods
    from() - Array-like object se nayi array banata hai.
    isArray() - Check karta hai ki value array hai ya nahi.
    Array.of() - Ek nayi array banata hai.
    keys() - Array ke keys ka iterator deta hai.
    values() - Array ke values ka iterator deta hai.
    entries() - Key-value pairs ka iterator deta hai.
